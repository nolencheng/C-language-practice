# C 語言練習器

互動式 C 語言學習與練習平台，涵蓋基礎到進階的題庫，適合初學者快速上手與自我評估。

## 線上使用

**https://nolencheng.github.io/C-language-practice/**

## 功能特色

- **10 大主題分類**：基礎語法、資料型別、運算子、控制流程、函式、陣列、指標、字串、結構體、記憶體管理
- **三種難度**：初級 / 中級 / 進階，或全部混合
- **兩種題型**：選擇題 + 填空題
- **即時解析**：每題作答後顯示詳細解說
- **學習記錄**：本地儲存各分類的答題正確率
- **錯題回顧**：練習結束後列出所有答錯的題目與解說
- **隨機出題**：每次練習題目順序不同，避免記憶題序

## 本地開發

直接開啟 `index.html` 即可，無需安裝任何依賴。

```bash
# 或使用任意靜態伺服器
npx serve .
python3 -m http.server 8080
```
